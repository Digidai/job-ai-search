# OpenJobs AI - AI-Powered Job Search Assistant

OpenJobs AI is an intelligent job search platform that leverages artificial intelligence to help job seekers find their dream careers. The platform provides personalized job recommendations, automated application processes, and smart career insights.

- **AI-Powered Job Matching**: Intelligent algorithms analyze your profile and preferences to find the perfect job opportunities
- **Smart Resume Analysis**: Automatic resume parsing and optimization suggestions
- **Automated Application Process**: Streamlined job application workflow with AI assistance
- **Career Insights**: Data-driven insights about job market trends and career paths
- **Personalized Dashboard**: Customized interface showing relevant job opportunities and application status
- **Mobile Responsive**: Fully responsive design for seamless experience across all devices

## üõ†Ô∏è Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Digidai/job-ai-search.git
cd job-ai-search
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
